### Svelte

Svelte works with Effector out of the box, no additional packages needed. 
Example:

```js
<script>
  import { createEvent, createStore } from 'effector'

  const increment = createEvent()
  const decrement = createEvent()
  const reset = createEvent()

  const counter = createStore(0)
    .on(increment, curr => curr + 1)
    .on(decrement, curr => curr - 1)
    .reset(reset)

  $: console.log($counter)
</script>

<div class="app">
  <div>Counter: {$counter}</div>
	<button on:click={increment}>Increment</button>
  <button on:click={decrement}>Decrement</button>
  <button on:click={reset}>Reset</button>
</div>
```

See ["Word Chain"](https://github.com/today-/citycatch) game application written with Svelte and Effector.
